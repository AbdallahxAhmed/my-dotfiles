# توثيق: تحويل مؤشرات Windows إلى Linux باستخدام win2xcur

هذا التوثيق يوضح كيفية تحويل حزم مؤشرات ويندوز (التي تحتوي على ملفات `.cur` و `.ani` وملف `Install.inf`) لتعمل بشكل أصلي على بيئات لينكس مثل KDE Plasma، مع حل مشكلة عدم تعرف الأداة على المؤشرات بسبب أخطاء ملف التثبيت.

### المتطلبات المسبقة
التأكد من تثبيت أداة التحويل الأساسية `win2xcur` (لمستخدمي Arch/CachyOS عبر AUR):
```bash
yay -S win2xcur
```

---

### الخطوة 1: تصحيح أخطاء ملف `Install.inf`
بعض المصممين يكتبون ملف `Install.inf` بصيغة تسبب خطأ (مثل `KeyError` أو `ValueError`) عند قراءته بواسطة سكريبت التحويل.

1. افتح ملف `Install.inf` الموجود داخل مجلد المؤشرات باستخدام محرر نصوص (مثل `micro`):
   ```bash
   micro /path/to/cursor/Install.inf
   ```
2. ابحث أسفل قسم `[DefaultInstall]` عن السطر المسؤول عن إضافات الريجستري، والذي غالباً يكون مكتوباً بشكل خاطئ كالتالي:
   ```ini
   AddReg = Scheme.Reg,Wreg
   ```
3. قم بتعديله ليقرأ المصفوفة الصحيحة فقط (وهي عادةً `Scheme.Reg`) ليصبح:
   ```ini
   AddReg = Scheme.Reg
   ```
4. احفظ التعديل واخرج من المحرر.

---

### الخطوة 2: تهيئة بنية المجلدات الهدف
أداة التحويل لا تنشئ المجلدات تلقائياً في حال عدم وجودها. ويجب وضع الملفات النهائية في مسار ثابت مثل `~/.local/share/icons/` لتجنب ضياع الثيم عند تنظيف مجلدات مؤقتة مثل الـ Downloads.

أنشئ المجلد الأساسي للثيم وبداخله المجلد الفرعي `cursors`:
```bash
mkdir -p ~/.local/share/icons/<Theme_Name>/cursors
```
*(استبدل `<Theme_Name>` بالاسم الذي تريده للثيم، مثل `Win11-Concept-Dark`).*

---

### الخطوة 3: التحويل وإنشاء الروابط (Symlinks)
استخدم أداة `win2xcurtheme` لقراءة ملف التثبيت وتحويل الملفات وإنشاء الروابط الرمزية بأسماء لينكس الصحيحة. 

**ملاحظة هامة:** يجب توجيه الأمر إلى ملف `Install.inf` تحديداً، وليس إلى المجلد الذي يحتويه، ويجب أن يكون مسار الإخراج موجهاً لمجلد `cursors`:
```bash
win2xcurtheme /path/to/cursor/Install.inf -o ~/.local/share/icons/<Theme_Name>/cursors
```

---

### الخطوة 4: تعريف الثيم للنظام
لكي تظهر المؤشرات في إعدادات KDE Plasma (System Settings > Colors & Themes > Cursors)، يجب إنشاء ملف `index.theme` في المجلد الأساسي للثيم يصف محتواه.

نفذ هذا الأمر لإنشاء الملف مباشرة:
```bash
echo -e "[Icon Theme]\nName=<Theme_Name>\nComment=Custom Converted Cursor Theme" > ~/.local/share/icons/<Theme_Name>/index.theme
```

بمجرد اكتمال هذه الخطوة، أصبح الثيم جاهزاً ويمكن تطبيقه من واجهة النظام. يتم تكرار نفس الخطوات تماماً مع أي نسخة أخرى (مثل Light/Dark) مع تغيير مسارات وأسماء المجلدات.
